# OpenChakra (Historical Fork)

This repository was originally used as an experimental foundation during the early development of ForgeUI Studio.

Active development has now moved to:

* ForgeUI Studio
* ESP32-P4 UI Studio
* ForgeUI-P4
* ForgeUI-One

This repository is no longer actively maintained and now exists only as a historical reference/archive.

# ACTIVE PROJECT

Please use the active ForgeUI Studio repository instead:

```text
https://github.com/RTechAI/esp32p4-ui-studio
```

ForgeUI Studio has evolved far beyond the original OpenChakra workflow and now includes:

* LVGL export generation
* ESP32-P4 deployment workflows
* embedded HMI tooling
* live hardware flashing
* browser preview modes
* embedded touchscreen workflows
* industrial UI direction
